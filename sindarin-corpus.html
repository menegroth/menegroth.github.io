The Sindarin Corpus Organized by Grammatical Features

Here are all the currently-known Sindarin phrases and grammar examples. I’ve organized them by grammatical category so you can easily see what evidence there is for any given feature of the language. Within each section, the examples are ordered chronologically by when Tolkien wrote them. This shows how his conception of the language changed over time. For features

In some examples, inflections or mutations have changed the form of a word. In those cases, I’ve noted the uninflected form like this: Edain Adan.

As always, text marked like this is a best guess based on context and Tolkien’s other writings—in other words, not something Tolkien actually wrote.

This page was made possible by the list of phrases on Paul Strack’s wonderful Eldamo site.

Final a → ai

Supporting Examples

When followed by only a single consonant, a in a final syllable changed to ai in the plural.

  • Beraid ← Barad
  • cirth ← certh
  • crebain ← craban
  • Edain ← Adan
  • Erain ← Aran
  • Evair ← Avar
  • gevair ← gavar
  • Goerthaid ← Gorthad
  • levain ← lavan
  • Periain ← Perian hobbit
